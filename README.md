A simple cli tool to add your bookmarks locally 😎 <br>
It provide a text based interface and an interractive interface to store your webmarks<br>
List of operation that are supported<br>
<ul>
<li>[.] list -listing all the webmarks</li>
<li>[.] add - add a new webmark</li>
<li>[.]remove -remove a webmark</li>
<li>[.] update -update the url of webmark</li>
</ul>
