A simple cli tool to add your bookmarks locally 😎
It provide a text based interface and an interractive interface to store your webmarks
List of operation that are supported
[.] list -listing all the webmarks
[.] add - add a new webmark
[.]remove -remove a webmark
[.] update -update the url of webmark
