A simple cli tool to add your bookmarks locally 😎 <br>
It provide a text based interface and an interractive interface to store your webmarks<br>
List of operation that are supported<br>
<ul>
<li>[.] list -listing all the webmarks</li>
<li>[.] add - add a new webmark</li>
<li>[.]remove -remove a webmark</li>
<li>[.] update -update the url of webmark</li>
</ul>

<h1>Instruction to use</h1><br>
<b>Download the webmarkers.exe from the release 📩</b> <br>
➡️ Go to that directory where you downloded and open with cmd using "cmd." <br>
➡️ it will open in command line ,br>
To add a new link, run the following command:

```bash
webmarkers link add -n "Name" -u "URL" -c "Category"
```
there are initially three categories 1)anime 2)learning 3)movie <br>
To move into interactive mode

```bash
webmarkers interactive
```
